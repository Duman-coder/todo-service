# 📝 todo-service

Минималистичное REST API приложение для управления задачами (To-Do), построенное с использованием Java и Spring Boot.

## 🚀 Функциональность

- CRUD-операции над задачами (создание, чтение, обновление, удаление)
- Фильтрация и пагинация задач
- Асинхронная обработка через Spring WebFlux
- Хранение данных в PostgreSQL
- Хранение истории и файлов в MongoDB (с использованием GridFS)
- Ограничение MIME-типа и размера загружаемых файлов
- Миграции с Flyway
- Разделение на DTO, модель, сервис, репозиторий и контроллер


## 🛠️ Стек технологий

- Java 21
- Spring Boot 3
- Spring WebFlux
- Spring Data JPA
- Spring Data MongoDB
- PostgreSQL
- MongoDB + GridFS
- Lombok
- Gradle (Kotlin DSL)
- Flyway
